웹브라우저의 종류

인터넷 익스플로러
마이크로소프트에서 만든 브라우저로 과거 국내 시장에 가장 많이 사용
FireFox
Mozilla재단에서 오픈 소스로 만들 것으로 w3c의 웹 표준을 선도한다.
Chrome
구글에서 만든 것으로 좋은 디버거를 갖추고 있어 디버깅에 많이 사용된다.

언어 및 framework

ajax
Ajax(Asynchronous JavaScript and XML, 에이잭스)는 비동기적인 웹 애플리케이션의 제작을 위해 아래와 같은 조합을 이용하는 웹 개발 기법이다.
vue
Vue.js(간단히 Vue, /vjuː/, 뷰/view)는 웹 애플리케이션의 사용자 인터페이스를 만들기 위해 사용하는 오픈 소스 프로그레시브 자바스크립트 프레임워크이다.
springframework
스프링 프레임워크(영어: Spring Framework)는 자바 플랫폼을 위한 오픈 소스 애플리케이션 프레임워크로서 간단히 스프링(Spring)이라고도 한다.
